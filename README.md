# Atlas

Atlas

Atlas is a lightweight, high-performance HTTP server written in modern C++, built from scratch to explore low-level networking, event-driven architecture, and clean system design.

⸻

🚀 Overview

Atlas is designed as a learning-focused yet production-inspired project, implementing:
	•	Non-blocking sockets
	•	Event-driven I/O (kqueue on macOS, epoll on Linux)
	•	Incremental HTTP parsing
	•	Clean abstraction layers (EventLoop, Server, Parser, Routing)

The goal is to understand how real-world servers (like Nginx) work under the hood.
